
## Разработка информационной системы для сети фотосалонов "Блиц-фото"

Проект представляет собой полнофункциональное приложение для управления фотосалоном, включающее систему учета услуг, сотрудников и финансов. Приложение разработано с использованием Python и фреймворков PySide6 (GUI) и PostgreSQL (база данных), а также управление ей через DBeaver.

### Основные компоненты приложения:

#### Backend:
- **PostgreSQL**: Используется для хранения данных о сотрудниках, услугах, заказах и финансовой отчетности.
- **Python** (psycopg2): Библиотека для взаимодействия с PostgreSQL.
- **bcrypt**: Для шифрования паролей пользователей.

#### Frontend:
- **PySide6**: GUI-интерфейс приложения.
- **QLineEdit**, **QPushButton**, **QTableWidget**: Элементы графического интерфейса.
- **QFileDialog**: Диалоговые окна для выбора файлов изображений.

#### Функциональные возможности:
- Авторизация пользователей с проверкой учетных записей.
- Управление списком сотрудников и услуг студии.
- Создание и редактирование заказов клиентов.
- Генерация финансовых отчетов и зарплатных ведомостей.
- Экспорт отчетов в CSV-формате.

### Требования к среде разработки:
- Python 3.x
- PySide6 (`pip install pyside6`)
- PostgreSQL сервер с установленной базой данных
- `psycopg2` библиотека (`pip install psycopg2-binary`)
- `bcrypt` библиотека (`pip install bcrypt`)

### Установка и запуск приложения:

1. **Установка зависимостей**:
```bash
pip install -r requirements.txt
```

2. **Создание базы данных**:
Создать базу данных PostgreSQL с именем `postgres`, используя конфигурационные настройки в коде.

3. **Настройка окружения**:
Скрипт ожидает соединения с локальной базой данных. Проверьте конфигурацию базы данных в файле `project.py`.

4. **Запуск приложения**:
```bash
python src/project.py
```
